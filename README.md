# openhandstest

This repository contains:
1. A simple FizzBuzz implementation in Python.
2. A simple TODO CLI tool in Python.

## How to run

### FizzBuzz
```bash
python3 fizzbuzz.py
```

### TODO CLI
```bash
# Add a task
python3 todo.py add "Task name"

# List tasks
python3 todo.py list

# Remove a task
python3 todo.py remove "Task name"
```
